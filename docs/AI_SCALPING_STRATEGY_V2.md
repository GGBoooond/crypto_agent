# AI Scalping Strategy V2 (AI 剥头皮策略 V2)

## 1. 策略概述

本策略专为加密货币高频/短线交易设计，旨在利用**微观市场结构**捕捉短期价格波动（0.5% - 2.0%）。

核心理念：**"硬数据驱动，软逻辑决策"**
- **硬数据 (Python)**: 使用 `pandas` 和 `numpy` 实时计算机构级技术指标，确保数据的绝对客观性和精确性。
- **软逻辑 (AI)**: 利用 DeepSeek (IQ 160+) 分析指标间的复杂关系、市场情绪和形态组合，做出最终交易决策。

---

## 2. 核心技术指标

我们不再让 AI "看图说话"，而是直接喂给它精确计算的数学指标：

### A. 趋势判断 (Trend)
- **EMA 50 (50周期指数移动平均线)**: 
  - 价格 > EMA 50 => **看涨趋势** (只做多或观望)
  - 价格 < EMA 50 => **看跌趋势** (只做空或观望)
  - *作用*: 过滤逆势交易，提高胜率。

### B. 入场时机 (Timing)
- **RSI 14 (相对强弱指数)**:
  - RSI < 30: **超卖** (潜在做多机会)
  - RSI > 70: **超买** (潜在做空机会)
  - *作用*: 捕捉价格回归均值的时机。
- **Bollinger Bands (布林带, 20, 2)**:
  - 价格触及下轨: 支撑位 (做多信号)
  - 价格触及上轨: 阻力位 (做空信号)
  - *作用*: 利用统计学原理判断价格是否偏离正常范围。

### C. 动能确认 (Momentum)
- **MACD (12, 26, 9)**:
  - 柱状图 (Histogram) 翻红/翻绿: 确认趋势强度的变化。
  - *作用*: 防止在下跌趋势中过早抄底（接飞刀）。

### D. 风控指标 (Risk)
- **ATR (平均真实波幅)**:
  - *作用*: 动态计算止损和止盈距离。
  - 波动大时 -> 止损宽 (避免被震荡扫出)
  - 波动小时 -> 止损窄 (保护利润)

---

## 3. AI 决策逻辑 (Prompt 设计)

我们设计了严密的 "If-Then" 逻辑链，强制 AI 遵守交易纪律：

### 做多场景 (Long Setup)
1. **趋势回踩**: 上升趋势中，价格回调至 EMA20 或 布林中轨。
2. **超卖反弹**: 价格跌破布林下轨 + RSI < 30 + 出现阳线。
3. **突破确认**: 强力突破布林上轨 + 成交量放大 2倍以上。

### 做空场景 (Short Setup)
1. **趋势受阻**: 下跌趋势中，价格反弹至 EMA20 或 布林中轨。
2. **超买回调**: 价格突破布林上轨 + RSI > 70 + 出现阴线。
3. **跌破确认**: 强力跌破布林下轨 + 成交量放大。

---

## 4. 交易频率与执行 (重要!)

**关于 "2分钟轮询" 的局限性**:
- 剥头皮策略的核心是**速度**。
- 加密货币市场波动极快，2分钟的延迟足以错过最佳入场点，甚至导致在最高点接盘。
- **现状**: 极度低频，因为 2分钟一次大概率处于"无信号"的中间状态。

**建议改进架构 (V3 规划)**:
1. **WebSocket 实时数据**: 毫秒级接收最新成交价 (Ticker) 和 K线更新。
2. **本地预计算**: 每收到一个 Tick，本地 Python 脚本立即更新指标 (RSI, Bollinger)。
3. **事件驱动触发**: 只有当 `RSI < 30` 或 `Price < LowerBand` 时，才调用 AI 进行二次确认。
   - *优势*: 既保证了全天候毫秒级监控，又节省了 AI Token 费用。

---

## 5. 风控参数

| 参数 | 值 | 说明 |
| :--- | :--- | :--- |
| **止损 (Stop Loss)** | 动态 (1.5 - 2.0 x ATR) | 根据当前市场波动率自动调整 |
| **止盈 (Take Profit)** | 动态 (2.0 - 3.0 x ATR) | 盈亏比至少 1.5:1 |
| **单笔风险** | 账户余额的 1% - 2% | 严格控制单次亏损 |
| **最大持仓时间** | 15 - 30 分钟 | 剥头皮不恋战，时间到必须平仓 |

V2 策略将 AI 从一个"看图说话的评论员"变成了一个"拿着精密仪器的狙击手"。它不再凭感觉交易，而是基于严格的数学概率模型进行博弈。
